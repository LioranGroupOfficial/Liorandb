# LioranDB Server API Documentation

## Overview

This document describes all available REST API endpoints for **LioranDB Server**.
All database, collection, and document routes require **Bearer JWT authentication**.

---

## Auth Endpoints

### **Register**

**POST /auth/register**
Creates a new user.

**Body:**

```json
{ "username": "string", "password": "string" }
```

**Response:**

```json
{ "user": { "id": "string", "username": "string" }, "token": "jwt" }
```

### **Login**

**POST /auth/login**
Authenticates a user.

**Body:**

```json
{ "username": "string", "password": "string" }
```

**Response:**

```json
{ "user": { "id": "string", "username": "string" }, "token": "jwt" }
```

---

## Health Check

### **GET /health**

Returns server status.

**Response:**

```json
{ "ok": true, "time": "ISO timestamp" }
```

---

## Database Endpoints

All require authentication.

### **List Databases**

**GET /databases**

**Response:**

```json
{ "databases": ["db1", "db2"] }
```

### **Create Database**

**POST /databases**

**Body:**

```json
{ "name": "databaseName" }
```

**Response:**

```json
{ "ok": true, "db": "databaseName" }
```

### **Delete Database**

**DELETE /databases/:db**

**Response:**

```json
{ "ok": true }
```

---

## Collection Endpoints

Authenticated.

Route prefix: **/db/:db/collections**

### **List Collections**

**GET /db/:db/collections**

**Response:**

```json
{ "collections": ["users", "posts"] }
```

### **Create Collection**

**POST /db/:db/collections**

**Body:**

```json
{ "name": "collectionName" }
```

**Response:**

```json
{ "ok": true }
```

---

## Document Endpoints

Authenticated.

Route prefix: **/db/:db/collections/:col**

### **Insert Document**

**POST /db/:db/collections/:col**

**Body:** Any JSON document.

**Response:**

```json
{ "ok": true, "doc": { ...document } }
```

### **Find Documents**

**POST /db/:db/collections/:col/find**

**Body:**

```json
{ "query": { ... } }
```

**Response:**

```json
{ "results": [ ...docs ] }
```

### **Get Document By ID**

**GET /db/:db/collections/:col/:id**

**Response:**

```json
{ "doc": { ... } }
```

### **Update Document**

**PATCH /db/:db/collections/:col/:id**

**Body:** Fields to update.

**Response:**

```json
{ "updated": true }
```

### **Delete Document**

**DELETE /db/:db/collections/:col/:id**

**Response:**

```json
{ "ok": true }
```

---

## Authentication Header

All protected routes must include:

```
Authorization: Bearer <token>
```

---

## Notes

* All JSON bodies require `Content-Type: application/json`.
* `_id` is automatically generated by LioranDB.
* Queries support basic object matching like MongoDB-style filters.
